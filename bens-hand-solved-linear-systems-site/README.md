# Ben's Hand-Solved Linear Systems

A static one-page website draft for **Ben's Hand-Solved Linear Systems**. Includes a quick FAQ section, pricing estimator, services, medium menu, and static request form.

## Files

- `index.html` — the website markup
- `styles.css` — all styling
- `script.js` — small pricing calculator + form demo message
- `assets/logo.png` — generated logo

## How to view locally

Open `index.html` in a browser, or run a small local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Notes

The request form is a front-end mockup. To make it real, connect it to Netlify Forms, Formspree, a custom backend, or an email service.
